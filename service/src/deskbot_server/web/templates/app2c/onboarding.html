歪BRUFIKSETUP

先配置大模型

小歪的 AI 表情设计 和 对话 都由大模型驱动。配置一次火山方舟（Ark）的密钥，之后所有功能就能用了。整个过程大约 1 分钟。

获取火山方舟密钥

在火山方舟控制台完成两件事，拿到下一步要填的两个值：

  1. 开通并创建 API Key（即 ARK_API_KEY）。
  2. 创建一个推理接入点 Endpoint，复制它的 ID（形如 ep-xxxxxxxx）作为「模型 ID」。
  3. 控制台入口：console.volcengine.com/ark ↗

已经有 API Key 和 Endpoint ID？直接进入下一步填写即可。

填写并测试

填入上一步拿到的 Endpoint ID 和 API Key，点「测试连接」确认可用后保存。
