先配置 AI 能力

小歪的 AI 表情设计、图片生成 和 对话 都由火山方舟驱动。你只需填一个 API Key，用到哪个模型我们会自动帮你路由好。
🔒 密钥只保存在你本机，不会上传云端。

获取火山方舟 API Key

你只需要一个 API Key —— 用到哪个模型我们会自动帮你选好（可切换），无需创建推理接入点。

  1. 打开火山方舟控制台，开通并创建 API Key（即 ARK_API_KEY）。
  2. 在控制台「开通管理」里开通模型 doubao-seed-2-1-pro（首次使用需开通，对话/表情/图片都用它）。
  3. 控制台入口：console.volcengine.com/ark ↗

已经有 API Key？直接进入下一步粘贴即可。

粘贴 API Key

填入火山方舟 API Key，点「测试连接」确认可用后保存即可。模型已自动选好，无需填写。

🔒 只保存在你本机（服务端 .env），不会上传云端
